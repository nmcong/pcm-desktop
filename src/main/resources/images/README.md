# Images Directory

Place your application images and icons here.

## Required Files

- `app-icon.png` - Application icon (recommended size: 256x256 or 512x512)

## Optional Files

- Additional icons
- Splash screen images
- UI assets

## Format Recommendations

- **PNG** for icons and UI elements
- **SVG** for scalable graphics
- **JPG** for photos

## Icon Sizes

- **16x16** - Small icon
- **32x32** - Standard icon
- **48x48** - Medium icon
- **256x256** - Large icon
- **512x512** - HD icon
